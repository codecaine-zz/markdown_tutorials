## üîç Quick Find Guide

### Looking for...

**A command-line tool?** ‚Üí `cli-tools/[category]/`
- File operations ‚Üí `file-management/`
- System info ‚Üí `system-monitoring/`
- Text editing ‚Üí `editors/` or `text-processing/`
- Network tasks ‚Üí `network-tools/`
- Security/pentesting ‚Üí `security-tools/`

**Programming language basics?** ‚Üí `programming/[language]/`
- Python ‚Üí `programming/python/python-complete-guide.md`
- JavaScript ‚Üí `programming/javascript/runtimes/`
- PHP, Rust, TypeScript, V ‚Üí `programming/[language]/[language]-complete-guide.md`

**Language-specific packages/frameworks?** ‚Üí `programming/[language]/[category]/`
- JavaScript packages ‚Üí `programming/javascript/packages/`
- Python packages ‚Üí `programming/python/[web-frameworks|gui|data-science|utilities]/`

**System configuration?** ‚Üí `system/[os]/`
